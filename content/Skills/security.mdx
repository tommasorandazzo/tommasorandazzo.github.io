Keeping data secure should be a top priority for any business. No one deserves to have their information sitting on a random website's insecure server for the taking. It is our responsibility as developers and site maintainers to provide a secure site from end-to-end.

Using the latest authorization workflows and leveraging services like <a href="https://www.cloudflare.com/">Cloudflare</a> provide us with a leg up from inflitrators to keep information where it should be.
